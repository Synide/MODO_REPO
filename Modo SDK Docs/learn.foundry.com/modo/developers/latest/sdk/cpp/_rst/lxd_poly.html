LxResult Validate(LXtObjectID self, LXtObjectID mesh)¶

Subdivision object provides the interface to use external subdivision engine for PSUB polygon type. The PSUB polygon type calculates subdivision points by this object while it is set instead of internal methods. Generates topological net for subdivision surface with cage polygons in the given mesh.
